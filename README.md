# 🎧 Headphone 3D Landing Page

Experience sound like never before — a cinematic, scroll-driven UI built for the future of spatial audio.

---

## 🌐 Live Demo

👉 https://headphone-3d.vercel.app/
## 🚀 Overview

This project is a **futuristic product landing page** that showcases a next-generation spatial audio headphone using immersive animations and storytelling.

It features a **scroll-based cinematic experience**, where the product dynamically transitions through:

* Hero introduction
* Disassembly (engineering view)
* Internal chip reveal
* Reassembly
* Final product showcase

Designed with an **Apple-level UI philosophy**, the goal is to create a visually engaging and premium user experience.

---

## ✨ Features

* 🎬 Scroll-driven storytelling
* 🎥 224-frame image sequence animation
* ⚡ Smooth animations using Framer Motion
* 🎯 Section-based transitions with precision timing
* 🌌 Immersive dark-themed UI
* 🧠 Performance-optimized rendering
* 📱 Responsive design

---

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Balamurugan1503/headphone_3d.git
cd headphone_3d
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open in browser:

```bash
http://localhost:3000
```

---

## 🌐 Deployment

This project is deployed using **Netlify**.

To deploy:

1. Push code to GitHub
2. Connect repository to Netlify
3. Set build command:

   ```
   npm run build
   ```
4. Publish directory:

   ```
   .next
   ```
